## 🚀 Forma Más Fácil

### Paso 1: Abrir Terminal en `codigo/`

```bash

```

### Paso 2: Generar Insights

```bash

```

**Esperado:**
- Lee 3,804 puertos + 157,604 emisiones
- Genera 4 CSV en `../insights/`

```bash

```

**Esperado:**
- Generador 7 PNG + 1 TXT
- Guarda en `../resultados/`
- Total: 8 archivos nuevos

## ⚠️ Solución de Problemas

| Error | Solución |
|-------|----------|
| ModuleNotFoundError: analyzer_spacehack | Estás en la carpeta equivocada. Ve a `codigo/` |
| FileNotFoundError: datasets/ | Los archivos están en `../datasets/` |
| Gráficos blancos | Ejecuta `uso_datos_puertos.py` primero |
| Imports no funcionan | Cierra Python e intenta de nuevo |
